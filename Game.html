Le Jeu du Pendu avec python

Notre projet consiste créer avec python le Jeu du Pendu, dont le but est de deviner un mot choisi aléatoirement. Le joueur doit entrer au clavier des lettres et a 10 essais pour trouver le mot. Si le joueur commet plus de 10 erreurs, la partie est perdue.
